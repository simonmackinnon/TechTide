# Project TechTide

## Scope

- E-commerce website

  * User management
  * Product Catalog
  * Shopping Cart
  * Order Check-out (mock payment) 

## Tech Stacks

- Front-end: React
- Back-end: Python Flask
- Database: PostgreSQL

## Setup

- Install Docker Desktop
- Install kubctl and Helm
- Install AWS cli and Terraform
- Install minikube
- Create git repo (this)
